# RAG Knowledge Chatbot

This project demonstrates a Retrieval-Augmented Generation (RAG) chatbot built using LangChain, FAISS, and Groq LLM.

## Features
- Retrieval Augmented Generation (RAG)
- FAISS vector database
- HuggingFace embeddings
- LangChain pipeline
- Groq Llama 3 LLM

## Technologies
- Python
- LangChain
- FAISS
- HuggingFace
- Groq LLM
- Sentence Transformers

## Dataset
SQuAD dataset from HuggingFace

## Workflow

Dataset → Text Chunking → Embeddings → FAISS Vector DB → Retriever → LLM → Answer
